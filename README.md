# MipyGame Engine Pro

**MipyGame** — это кастомный 2D-игровой движок на Python с графическим интерфейсом (IDE). 

## 🎯 Философия и Архитектура
Движок базируется на парадигме **Construct Classic (Scirra)**. Мы уходим от концепции префабов Unity и разделяем сущности на:
1. **Object Types (Типы объектов):** Глобальные шаблоны, хранящие графику, настройки физики и Скрипты-Поведения (Behaviors).
2. **Instances (Экземпляры):** Конкретные объекты на холсте, которые лишь ссылаются на свой Тип, но имеют уникальные координаты (X, Y), имя и слой отрисовки (Z-Index).

## 🛠 Технологический стек
- **Core / Render:** Pygame
- **UI / IDE:** PyQt6 (модульная система `QDockWidget`)
- **Physics:** Pymunk 2D

## 🚦 Как запустить проект
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите редактор: `python main.py`
3. Запустите тесты: `python -m unittest test_engine.py`
